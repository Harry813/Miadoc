# 游戏指令

<tldr>
    <p>关键词: <shortcut>game</shortcut> <shortcut>Game</shortcut> <shortcut>GAME</shortcut></p>
</tldr>

## 介绍

不带参数传入将显示目前支持的[游戏列表](#game_list)。

## 开启游戏

命令：`@Mia game <游戏名>`

开启指定游戏。

> 该命令仅限管理员使用。
{style="note"}

## 结束游戏

命令：`@Mia game end`

强制结束当前游戏，并清理数据库

> 该命令仅限管理员使用。
{style="note"}
